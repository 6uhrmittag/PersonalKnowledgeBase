# Windows Subsystem for Linux (WSL/WSL2)

## Notes

### Ubuntu 24.04 LTS supports Cloud-init

tl;dr: Create this folder and a Cloud-Init file per Distro: `C:\Users\$env:USERPROFILE\.cloud-init\Ubuntu-24.04.user-data`
